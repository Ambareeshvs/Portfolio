# Portfolio Website

Simple responsive portfolio site built with vanilla HTML, CSS, and JS.
Link: https://portfolio-ambareesh.vercel.app/

## Run locally

Open `index.html` directly in a browser, or serve it with any static server.

## Customize

- Update email in `index.html` (`mailto:...`).
- Update GitHub username in `script.js` (`GITHUB_USERNAME`).
- Update career journey, education, and certifications in `script.js` (`PROFILE_DETAILS`).
- Use the header toggle to switch dark/light mode (saved in browser local storage).

## Data source

- GitHub profile stats and latest repositories are loaded from the public GitHub API at runtime.
